### Solution 1 - Temporal try to install it in the conatiner
Run this inside the container
```bash
apt-get update
apt-get install -y libtiff-dev libtbb-dev
```
Then I can run
```bash
mkdir build
cd build
cmake ..
make
```
Without any problem.
